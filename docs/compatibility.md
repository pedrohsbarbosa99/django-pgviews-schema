| -------------- | --------------------- |
| 3.2            | 0.0.1                 |
| 4.0            | 0.0.1                 |
| 4.1            | 0.0.3                 |
| 4.2            | 0.0.4                 |
| 5.0            | 0.0.4                 |

# Python 3 Support

Django PGViews Redux officially supports Python 3.7+ and may work on 3.6, but there are no guarantees.
